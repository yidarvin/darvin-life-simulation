# Life is a Simulation

An idle/clicker game about pretending you know what you're doing. CS career simulator with sarcastic humor.

Built with React + Vite + Tailwind. Deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:5173.
